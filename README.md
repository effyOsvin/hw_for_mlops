# mlops

## Task

We solve the problem of image classification using a dataset MNIST .

## Usage

### Setup

```
poetry install
pre-commit install
pre-commit run -a
```

### Run experiments

For train the model and save it afterward, run:

```
python hw_for_mlops/train.py
```

For infer a model, run:

```
python hw_for_mlops/infer.py
```
